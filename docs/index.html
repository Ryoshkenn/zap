zap

A terminal launcher for AI coding CLIs.
Pick a folder, pick a provider, press enter. That's it.

# Homebrew
brew install Ryoshkenn/zap/zap
# Homebrew on Linux
brew install Ryoshkenn/zap/zap

Don't have Homebrew? Install it here — it works great on Linux too.

# Scoop
scoop bucket add zap https://github.com/Ryoshkenn/scoop-zap
scoop install zap
# Go 1.21+
go install github.com/Ryoshkenn/zap/cmd/zap@latest

Why zap

⚡

One keystroke

Type zap, pick a folder, pick a provider. Done. No more cd + long flag combos.

🔍

Smart recents

Favorites and recently used folders float to the top. Your most-used repos are always one keypress away.

🚀

Direct exec on Unix

On macOS and Linux, zap execs into the CLI — zap disappears, the tool owns the TTY directly.

🪟

Windows native

Full support via Scoop. Forwards stdin/stdout/stderr and exit codes cleanly.

🎛️

Per-provider flags

Toggle flags like --dangerously-skip-permissions from the picker. No muscle memory needed.

🔧

Extensible

Add any CLI tool via a single YAML entry in ~/.config/zap/config.yaml. No Go required.
